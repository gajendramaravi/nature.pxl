Nature Photographer · Chhattisgarh, India

I'm a self-taught nature photographer based in Chirmiri, Chhattisgarh — a land of dense forests, wild rivers, and untouched skies. Photography found me during long walks through the Achanakmar forests, where I learned that nature reveals itself only to those willing to be still and patient.

Every frame I capture is a conversation — between light and shadow, between stillness and motion. From macro drops on spider webs at dawn to storm clouds rolling over the Bastar plateau, I chase the moments that words can't hold.

GitHub Hosting Guide
Step 1 — Create photos.json
In your GitHub repo, create a file called photos.json containing your photo entries. Each photo needs: title, cat, loc, img (Google Photos lh3 URL), and h (height).
[
  {
    "title": "Mogra in My Garden",
    "cat": "macro",
    "loc": "Chirmiri, CG",
    "img": "https://lh3.googleusercontent.com/...",
    "h": 320
  }
]
Step 2 — Get raw URL
Go to your GitHub repo → click the file → click Raw. Copy the raw URL. It will look like:
https://raw.githubusercontent.com/username/repo/main/photos.json
Step 3 — Load into gallery
Paste the raw URL below and click Load. All your photos will appear in the main gallery above with filtering.
